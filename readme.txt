pour faire fonctionner le Projet il faut :

utiliser la commande suivante dans le dossier backend : npm i

Puis créer un dossier images/postImg.

Lancer le back avec la commande : nodemon server

créer un fichier .env avec les parametres suivant : 
SECRET_KEY=
SERVER_PORT=
BCRYPT_SALT_ROUND=
DBCONNECT=

utiliser la commande suivante dans le dossier frontend : npm install

npm install @craco/craco@alpha puis 

Lancer le front avec la commande : npm run starto
