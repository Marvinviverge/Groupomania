Pour faire fonctionner le Projet vous devez :

* Cloner le repo Github.

* Par le biais du terminal, situez vous dans le dossier Backend et utilisez la commande suivante: npm i

* Puis créer un dossier images/postImg.

* Créer un fichier .env avec les parametres suivant : 
    SECRET_KEY=
    SERVER_PORT=
    BCRYPT_SALT_ROUND=
    DBCONNECT=

* Vous pouvez maintenant lancer le back avec la commande : nodemon server

Si la procédure s'est bien déroulée, le terminal doit vous afficher un message "This server is running on port ****. Enjoy !"
Vous pouvez continuer la procédure de lancement du projet.

* Par le biais d'un nouveau terminal, situez vous dans le dossier Frontend et utilisez la commande suivante : npm install

* Puis, utilisez la commande suivante: npm install @craco/craco@alpha 
 
* Vous pouvez maintenant lancer le front avec la commande : npm run starto

Il se peut que le terminal affiche un message comme celui-ci:

    " ? Something is already running on port 3000.

    Would you like to run the app on another port instead? » (Y/n) "

Si c'est le cas, il vous suffit d'appuyer sur la touche y. Si la procédure, le terminal doit vous indiquer "Compiled successfully".

Vous pouvez maintenant vous rendre sur le site de Groupomania à l'adresse indiquée.
